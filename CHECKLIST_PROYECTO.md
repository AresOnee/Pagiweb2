## Proceso de Desarrollo

### Fase 1: Inicio
- [ ] Presupuesto aprobado por el cliente
- [ ] Anticipo recibido (50%)
- [ ] Reunión de kickoff (opcional)
- [ ] Contenido recibido del cliente

### Fase 2: Desarrollo
- [ ] Diseño y estructura del sitio
- [ ] Desarrollo de páginas
- [ ] Implementación de funcionalidades
- [ ] Carga de productos en el catálogo
- [ ] Configuración del formulario de contacto

### Fase 3: Pruebas
- [ ] Pruebas en diferentes navegadores
- [ ] Pruebas en dispositivos móviles
- [ ] Verificación de formularios
- [ ] Optimización de velocidad
- [ ] Revisión de ortografía y contenido

### Fase 4: Revisión del Cliente
- [ ] Presentación del sitio al cliente
- [ ] Recolección de feedback
- [ ] Implementación de ajustes solicitados
- [ ] Aprobación final del cliente

### Fase 5: Lanzamiento
- [ ] Registro del dominio
- [ ] Configuración de DNS
- [ ] Deploy a producción
- [ ] Verificación post-lanzamiento
- [ ] Entrega de accesos al cliente
- [ ] Pago final (50% restante)

## Notas Importantes

1. **Contenido:** El desarrollo puede iniciar con contenido parcial, pero se requiere el contenido completo para la entrega final.

2. **Revisiones:** Se incluyen hasta 2 rondas de revisiones menores. Cambios mayores de alcance se cotizan aparte.

3. **Productos:** El presupuesto incluye hasta 50 productos. Productos adicionales tienen costo extra.

4. **Dominio:** El dominio queda registrado a nombre del cliente. Es 100% de su propiedad.

5. **Código:** El código fuente se entrega al finalizar. Puede modificarlo o migrarlo cuando lo desee.
