# Guia de Productos GelChile

## ¿Que hace GelChile?

GelChile es una empresa chilena con mas de 20 anos de experiencia que se dedica a un tema fundamental para la seguridad electrica: la **puesta a tierra**.

Cada edificio, casa, fabrica o torre de telecomunicaciones necesita un sistema que lleve la electricidad no deseada (fallas, cortocircuitos, rayos) de forma segura hacia la tierra. Sin esto, las personas pueden recibir descargas electricas y los equipos pueden danarse o incendiarse.

GelChile vende todos los materiales necesarios para instalar estos sistemas y tambien ofrece servicios de instalacion y medicion profesional. Su oficina esta en Cerrillos (Santiago) y su bodega en Calera de Tango.

---

## ¿Que es una puesta a tierra?

Imagina que la electricidad es como el agua: siempre busca el camino mas facil para llegar al suelo. Una puesta a tierra es exactamente eso: un camino seguro y controlado para que la electricidad no deseada llegue a la tierra sin hacer dano.

El sistema basico funciona asi:
1. Se entierra una **barra o electrodo** de metal (cobre o grafito) en el suelo
2. Se conecta con **cables de cobre** a la instalacion electrica del edificio
3. Las conexiones entre cables se hacen con **soldadura exotermica** (fusion permanente) o **conectores mecanicos**
4. Se puede agregar **aditivo** a la tierra para que conduzca mejor la electricidad
5. Finalmente, se **mide** la resistencia para verificar que el sistema funciona correctamente

Todo lo que vende GelChile son las piezas de este rompecabezas.

### 6. Soldadura Exotermica (13 productos)

**¿Para que sirve?** La soldadura exotermica es el metodo para unir cables de cobre de forma permanente. Funciona asi: se pone polvo de aluminio especial en un molde de grafito, se enciende con un chispero, y el polvo se derrite a altisima temperatura fusionando los cables de cobre en una sola pieza. Esta union es mas resistente y duradera que cualquier conector mecanico.

## ¿Como se relacionan los productos entre si?

Un proyecto tipico de puesta a tierra usa varios de estos productos juntos. Aqui van dos ejemplos:

### Ejemplo 1: Casa o edificio pequeno

1. Se clava una **Barra Toma Tierra** (BTT-001/002/003) en el suelo
2. Se conecta con cable de cobre al tablero electrico
3. Las uniones se hacen con **Conectores con Perno Partido** (CPP-001) o **soldadura exotermica** (CEX + PAR-004 + PAR-003 + PAR-002)
4. Se instala una **Camarilla de Registro** (CAM-001) sobre el punto de conexion
5. Se **mide** la resistencia (SRV-001) para verificar que cumple la norma

### Ejemplo 2: Planta industrial o torre de telecomunicaciones

1. Se instala un **Electrodo Electromagnetico** Hunter Energy (ELE-001 a ELE-008) segun la capacidad requerida
2. Se puede agregar **Barras Quimicas** (BQH-001/BQV-001) si el suelo tiene alta resistividad
3. Se aplica **Aditivo** (ADI-001/ADI-002) alrededor de los electrodos para mejorar la conductividad
4. Se tiende una **Malla de Fleje** de cobre (SRV-002) conectando todos los puntos
5. Todas las uniones se hacen con **Soldadura Exotermica** (CEX + molde + chispero + masilla)
6. Se instala un **Pararrayo** (PAR-001) en el punto mas alto
7. Los componentes de control van en **Tableros TIBOX** (TAB-001/TAB-002)
8. Se **mide** todo el sistema (SRV-001) y se entrega informe
